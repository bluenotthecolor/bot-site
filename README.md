# 🌸 Lana Bot Site

The official website for **Lana**, a feature-rich Discord bot designed to make managing and enjoying your Discord server easier.

Explore Lana's features, browse commands, get support, and stay updated with the latest improvements.

## ✨ Features

- 🏠 Beautiful and responsive homepage
- 📖 Complete command list
- 💬 Dedicated support page
- 📊 Dashboard
- ⚡ Fast and lightweight
- 📱 Mobile-friendly design
